# Docs Collection

**Purpose**: Stores feature announcements displayed on the documentation page.

## Structure

- `_id`: ObjectId
- `Title`: String - Feature title
- `Description`: String - Detailed description of the feature
- `Progress`: String - Current status (e.g., "Planned", "In Progress", "Completed")
